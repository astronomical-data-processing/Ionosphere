# 程序说明文件

这个画图代码写的很匆忙，有很多地方比较丑陋，不过总归是可以勉强做到自动画出一个数据文件的所有map，后续有时间可能会做优化。

首先，数据需要预处理一下：从网站拿到数据后，手动删除所有头文件的描述信息，删到和此目录下的data.txt保持一致格式即可。
其次，对代码中的两个主要函数的参数做一下说明：

# ''data_select''函数：

file='data.txt'    ：数据文件
lat_num=71         ：数据中lat的个数，比如代码当中所用的数据中lat的范围是：-87.5到87.5，分辨率为2.5度，那么lat_num的值就是：(87.5+87.5)/2.5+1=71, 其他数据依此类推。
lon_num=73,        ：数据中lon的个数，比如代码当中所用的数据中lon的范围是：-180到180，分辨率为5度，那么lon_num的值就是：(180+180)/5+1=73, 其他数据依此类推。
time_num=25        ：数据中采样时间的个数，比如代码当中所用的数据中每隔一个小时采样一次数据，那么一天下来就是25个小时，我代码中选择了只画24个小时，需要注意的是：有的数据是按两小时一次采样的，所以要看清楚。


''plot_tec''函数：
hour = 24          :你画几个小时数据这里就取几
